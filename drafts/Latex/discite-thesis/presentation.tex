\section{Introduction}
Discite(lat. "to learn") - is an interactive peer-to-peer teaching site, which
simulates the direct instruction teaching model
\\[10pt]
The platform (http://discite.info/) allows:
\begin{enumerate}
  \item pick a teacher based on your interests
  \item view rankings, add ratings
  \item screen sharing
  \item audio/video chat
  \item presentations
  \item doodling
\end{enumerate}
}
